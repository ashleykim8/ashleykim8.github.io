Interactive Google Calendar API Script

For this project, I led the development of a Python script designed as an interactive platform to communicate with the user's Google Calendar API. This script empowers users to manage their public-facing Google Calendar locally through their terminal, offering robust functionalities to add, modify, and query events without needing browser access.

As the team lead, my primary responsibility was to ensure the successful delivery of the project within established deadlines. This involved overseeing the development lifecycle, fostering clear communication, and actively contributing to the technical implementation. I spearheaded the integration of core API interactions, setting up robust authentication flows and designing efficient logic for event management.

A significant challenge arose when a team member proposed an ambitious chatbot integration for natural language event creation. While innovative, it became evident that this feature was outside the scope of our project timeline and resources. As team lead, I facilitated a critical re-evaluation of our priorities. We proactively dedicated some time to research its feasibility alongside our original objectives, establishing a clear understanding that its implementation would only be pursued if we completed our core functionalities significantly ahead of schedule. This experience was instrumental in honing my ability manage project scope, and lead a team towards focused, deliverable outcomes without compromising quality or deadlines.

Project Highlights & Technical Contributions

  • Secure API Integration: Implemented Google Calendar API authorization flows, ensuring secure and authenticated access to user calendars.
  • Event Management Logic: Developed core functionalities for adding new events, modifying existing event details (time, description, attendees), and deleting events.
  • Querying and Display: Created robust methods to query and retrieve calendar events based on various criteria (date range, keywords) and display them clearly in the terminal.
  • User Interaction Design: Designed the terminal-based interface for intuitive user input and clear output, allowing seamless interaction with calendar data.
  • Scope Management & Prioritization: Successfully managed project scope by identifying and de-prioritizing features that did not align with the project's core objectives and timeline, ensuring on-time delivery of essential functionalities.
